About DevBook

The Centralized Peer Portfolio Registry

Our Mission

DevBook was built to solve a common challenge in developer bootcamps, university computer science classes, and engineering teams: the lack of a unified space to discover, review, and inspect classmates' active projects. DevBook provides a clean, automated portal where developers can submit their creations, link their code repositories, and list their technical skills in a single, searchable index.

Core Features

  • ⚡ Real-time Tags Filtering: Search projects instantly by programming languages, frameworks, or database tags.
  • 🐙 GitHub Integration: Automatically fetch repositories and pre-populate deployment resources directly from your GitHub profile.
  • 🔑 Secure Social Authentication: Seamless, protected user signup and onboarding workflows using Google and GitHub OAuth.
  • 🎭 Premium Visual Design: Supports high-contrast light and dark modes, z-indexed account management dropdown menus, custom detail modals, and mouse-hover 3D tilt effects.

Platform Tech Stack

DevBook is powered by a modern, high-performance web stack:

Django Python Supabase PostgreSQL HTML5 & Vanilla CSS JavaScript OAuth 2.0
